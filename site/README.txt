# Site de Inscrição (Frontend)

1) Publique seu Google Apps Script como Web App e copie a URL (ex.: https://script.google.com/macros/s/AKfycbx.../exec).
2) No arquivo `index.html`, substitua `COLE_AQUI_SUA_URL_DO_APPS_SCRIPT` pela URL do seu Web App.
3) Abra `index.html` no navegador para testar localmente.
4) Publique no GitHub Pages (Settings -> Pages).

O botão "Baixar comprovante" gera um PDF com jsPDF após o envio.
